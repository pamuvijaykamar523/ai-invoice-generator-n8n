# 🤖 AI Invoice Generator using n8n

An AI-powered invoice automation workflow built with n8n.

## 🚀 Features

- Customer details collection using Google Forms
- Automatic data storage in Google Sheets
- Dynamic invoice number generation
- HTML invoice generation using JavaScript
- Automatic PDF conversion using PDF.co
- Email invoice delivery through Gmail
- End-to-end business workflow automation

## 🛠 Tech Stack

- n8n
- Google Forms
- Google Sheets
- JavaScript
- PDF.co API
- Gmail
- Google Drive

## 📌 Workflow

Google Form
↓
Google Sheets
↓
Generate HTML Invoice
↓
Convert HTML to PDF
↓
Send PDF via Gmail
